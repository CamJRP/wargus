Installation instructions for Wargus 2.1

Software Requirements

  • Stratagus 2.1 (the Wargus version MUST match the stratagus version) (required).

  • libpng (required).

  • zlib (required).

  • Either the original or expansion wc2 CD (not Battle.net edition).

Installation

Mac OS X users: Extract the "contrib/macosx.tgz" file to the top-level directory and read "Read Me.rtf".

  • Step 1, Windows:

    • run: 
      build.bat [cdpath] [outputdir]
      NOTE: outputdir is optional, the default is 'data.wc2'
      This will extract the data to outputdir.

  • Step 1, Unix/POSIX:

    • run: 
      ./build.sh -p [cdpath] [outputdir]
      NOTE: outputdir is optional, the default is 'data.wc2'
      This will extract the data to outputdir.

  • Step 2, Run:

    You may now start wargus by running the command: 

    stratagus -d outputdir
